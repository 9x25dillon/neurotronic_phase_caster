### Option 1: Local LaTeX Installation

If you have LaTeX installed (e.g., TeX Live, MiKTeX):

```bash
cd papers

pdflatex geometry_higher_dimensional_intelligence.tex  # Run twice for references
```

### Option 2: Overleaf (Recommended)

1. Go to [Overleaf](https://www.overleaf.com)
2. Create a new project → Upload Project
3. Upload `geometry_higher_dimensional_intelligence.tex`
4. Click "Recompile" to generate PDF

### Option 3: Docker

```bash
docker run --rm -v $(pwd):/data texlive/texlive pdflatex /data/geometry_higher_dimensional_intelligence.tex
```
